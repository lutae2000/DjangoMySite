DjangoMySite
----------
#### 1. 프로젝트 내용
> 게시판, 방명록, 회원등록 기능을 가지고 있습니다. Python의 웹 프레임워크 Django v2.0.6,
mysqlclient v1.3.12 와 django-mathfilters v0.4.0 를 사용했습니다.
